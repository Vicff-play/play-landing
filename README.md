# 🎵 Play

Plataforma que conecta músicos a clientes para contratação de apresentações em eventos.

---

## 🚀 Sobre o projeto

O **Play** é um aplicativo que facilita a contratação de músicos para eventos como:

- Casamentos  
- Festas  
- Eventos corporativos  
- Cultos e eventos religiosos  

A plataforma permite que clientes publiquem eventos e músicos enviem propostas, tornando o processo mais organizado, seguro e eficiente.

---

## ⚙️ Funcionalidades

- Criação de eventos  
- Envio de propostas por músicos  
- Aceitação e confirmação de propostas  
- Comunicação entre cliente e músico  
- Gestão completa da contratação  
- Integração com pagamento
